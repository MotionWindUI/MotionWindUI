# Spacing and Radius

The spacing system is based off of TailwindCSS's spacing scale. The spacing scale is a set of values that are used to create consistent spacing between elements. The spacing scale is based on a 4px grid, which allows for easy alignment of elements. It can be used for things like padding, margin, and positioning.

## Radius Tokens

Radius tokens are used to define the border radius of an element. The radius tokens are based on the spacing scale, and are used to create consistent border radius values.
